# Introduction
This repository is a curated collection of custom skills, prompts, and configurations for my favorite AI tools, including Claude AI and the Cursor Editor.

## How to use it?
To get started, navigate to the folder of the AI tool you want to configure and import the skill files into your local application.

Here is the repository's directory structure:
```
root/
├── claude/
│   ├── skills/
│   │   ├── first_skill.md
│   │   ├── .....
│   │   └── end_skill.md
│   └── other_folder_skills/
│       └── end_skill.md
│
├── cursor/
└── ....
```
